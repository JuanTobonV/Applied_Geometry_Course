# Curso de Geometría Aplicada

Repositorio para abordar los teoremas del curso de Geometría Aplicada de la Universidad Nacional de Colombia.

## Instalación de Dependencias

1. Crear un entorno virtual:
```bash
python -m venv venv
source venv/bin/activate  # En Linux/Mac
```

2. Instalar las dependencias:
```bash
pip install -r requirements.txt
```

## Contenido

- **Teorema de Brianchon**: Verificación computacional del teorema usando geometría proyectiva
